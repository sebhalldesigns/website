foxdbg

A lightweight, cross-platform implementation of the Foxglove Studio protocol, using parallel threads, atomics and JPEG compression to ensure high performance with minimal latency.

TBReAI

A Formula Student AI system implementing a full self-driving stack, including perception, navigation and control. Uniquely to this project, the system is implemented with a fully cross-platform architecture, opting for a monolithic approach that removes dependency on the ROS ecosystem (mainly intended for Ubuntu), while reducing latency and overhead introduced by realtime TCP communications.

  • Perception

    Using OpenCV and LibPCL, camera images and lidar point-clouds are processed to identify objects in the vehicle's environment.

  • Navigation

    An online EKF SLAM system continuously maps the vehicle's environment, adjusting its position and planning a smooth trajectory around the track.

  • Control

    Realtime vehicle control logic manages communication with the vehicle over a CAN bus, managing state machines for powertrain management and completion of competition scenarios.

  • Runtime

    A runtime abstraction layer provides a consistent interface for the self-driving system, allowing for seamless running on production hardware, alongside multiple simulation environments.

TBReSim

A cross-platform vehicle simulator for TBReAI, built with the Unity game engine and using shared memory and discrete time synchronisation to ensure high-performance, deterministic simulation.

NanoKit

A set of lightweight, cross-platform libraries for building hardware accelerated user interfaces using OpenGL.

  • NanoWin

    A windowing library for creating and managing windows, input and events. NanoWin provides backends for Win32, X11, macOS AppKit and WebAssembly HTML5 Canvas.

  • NanoDraw

    A library for optimised rendering of 2D graphics using OpenGL.

  • NanoView

    A library implementing a generic view model for building interfaces, including a library of common controls such as buttons, sliders and text inputs.

  • NKGen

    A code generator for automatic generation of C source files from XML file definitions and supporting logic files (similar to C#'s XAML and AXAML). Integrated into the CMake build system, NKGen will run automatically at build time.

Jetson Breakout

An PCB breakout board for the Nvidia Jetson AGX Orin, containing two CAN transceivers and 60 Ohm resistors with a Molex DuraClik connector. This allows for direct connection of a Jetson to a CAN bus, without the need for any additional hardware.

Jetson Shock Mount

A 3D printed shock mount for the Nvidia Jetson AGX Orin, allowing for secure mounting of the Jetson to a vehicle.

Perception System Mount

A 3D printed mount for a perception system, providing an integrated mount for a camera and lidar sensor, with adjustable angles for optimal positioning.
